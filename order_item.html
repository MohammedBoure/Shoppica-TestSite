1. Add New Order Item (Admin Only)

Endpoint: POST /order_items

Description: Creates a new order item for a specified order (admin only).

2. Get Order Item by ID

Endpoint: GET /order_items/

Description: Retrieves a specific order item by its ID (order owner or admin only).

3. Get Order Items by Order

Endpoint: GET /order_items/order/

Description: Retrieves all order items for a specific order (order owner or admin only).

4. Update Order Item (Admin Only)

Endpoint: PUT /order_items/

Description: Updates an order item's details (admin only).

5. Delete Order Item (Admin Only)

Endpoint: DELETE /order_items/

Description: Deletes an order item by its ID (admin only).

6. Get All Order Items (Admin Only)

Endpoint: GET /order_items

Description: Retrieves a paginated list of all order items (admin only).
