Addresses API Documentation

Interact with the Addresses API endpoints hosted at

Back to API Testing Hub

1. Add New Address

Endpoint: POST /addresses

Description: Adds a new address for the authenticated user. Requires authentication.

2. Get User's Addresses

Endpoint: GET /addresses/me

Description: Retrieves all addresses for the authenticated user. Requires authentication.

3. Get Default Address

Endpoint: GET /addresses/me/default

Description: Retrieves the default address for the authenticated user. Requires authentication.

4. Get Address by ID

Endpoint: GET /addresses/

Description: Retrieves a specific address by its ID (user must own it or be admin). Requires authentication.

5. Update Address

Endpoint: PUT /addresses/

Description: Updates a specific address (user must own it or be admin). Requires authentication.

6. Delete Address

Endpoint: DELETE /addresses/

Description: Deletes a specific address (user must own it or be admin). Requires authentication.

7. Get User Address Stats

Endpoint: GET /addresses/me/stats

Description: Retrieves address statistics for the authenticated user. Requires authentication.

8. Get Addresses by User (Admin Only)

Endpoint: GET /admin/addresses/user/

Description: Retrieves all addresses for a specific user (admin only). Requires admin authentication.

9. Delete Addresses by User (Admin Only)

Endpoint: DELETE /admin/addresses/user/

Description: Deletes all addresses for a specific user (admin only). Requires admin authentication.

10. Search Addresses (Admin Only)

Endpoint: GET /admin/addresses/search

Description: Searches addresses using a single query word across multiple fields (admin only). Requires admin authentication.

11. Get All Addresses (Admin Only)

Endpoint: GET /admin/addresses

Description: Retrieves a paginated list of all addresses (admin only). Requires admin authentication.

12. Get Overall Address Stats (Admin Only)

Endpoint: GET /admin/addresses/stats

Description: Retrieves overall address statistics (admin only). Requires admin authentication.

13. Get User Address Stats (Admin Only)

Endpoint: GET /admin/addresses/user//stats

Description: Retrieves address statistics for a specific user (admin only). Requires admin authentication.
