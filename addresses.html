Addresses API Documentation

Interact with the Addresses API endpoints hosted at

Back to API Testing Hub

1. Add New Address

Endpoint: POST /addresses

Description: Adds a new address for the authenticated user.

2. Get User's Addresses

Endpoint: GET /addresses/me

Description: Retrieves all addresses for the authenticated user.

3. Get Address by ID

Endpoint: GET /addresses/<address_id>

Description: Retrieves a specific address by its ID (user must own it or be admin).

4. Update Address

Endpoint: PUT /addresses/<address_id>

Description: Updates a specific address (user must own it or be admin).

5. Delete Address

Endpoint: DELETE /addresses/<address_id>

Description: Deletes a specific address (user must own it or be admin).

6. Get Addresses by User (Admin Only)

Endpoint: GET /admin/addresses/user/<user_id>

Description: Retrieves all addresses for a specific user (admin only).

7. Get All Addresses (Admin Only)

Endpoint: GET /admin/addresses

Description: Retrieves a paginated list of all addresses (admin only).
