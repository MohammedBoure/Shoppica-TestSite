1. Top Selling Products

Endpoint: GET /analytics/products

Description: Retrieves top-selling products. Requires a valid session and admin privileges.

2. Sales Statistics

Endpoint: GET /analytics/sales

Description: Retrieves sales statistics for a specified date range. Requires a valid session and admin privileges.

3. User Statistics

Endpoint: GET /analytics/users

Description: Retrieves user statistics for a specified date range. Requires a valid session and admin privileges.

4. Customer Retention Rate

Endpoint: GET /analytics/customer-retention

Description: Retrieves customer retention rate for a specified date range. Requires a valid session and admin privileges.

5. Product Performance Trend

Endpoint: GET /analytics/product-performance

Description: Retrieves performance trend for a specific product. Requires a valid session and admin privileges.

6. Discount Effectiveness

Endpoint: GET /analytics/discount-effectiveness

Description: Retrieves discount effectiveness metrics for a specified date range. Requires a valid session and admin privileges.
