Note: Endpoints marked with (*) require authentication. Admin-only endpoints require admin privileges.

1. Add New Review (*)

Endpoint: POST /reviews

Description: Creates a new review for a product (authenticated user or admin only).

2. Get Review by ID

Endpoint: GET /reviews/<review_id>

Description: Retrieves a specific review by its ID (public access).

3. Get Reviews by Product

Endpoint: GET /reviews/product/<product_id>

Description: Retrieves all reviews for a specific product (public access).

4. Update Review (*)

Endpoint: PUT /reviews/<review_id>

Description: Updates a specific review (review owner or admin only).

5. Delete Review (*)

Endpoint: DELETE /reviews/<review_id>

Description: Deletes a specific review (review owner or admin only).

6. Get All Reviews (* Admin)

Endpoint: GET /reviews

Description: Retrieves a paginated list of all reviews (admin only).

7. Search Reviews

Endpoint: GET /reviews/search

Description: Searches reviews by product ID, user ID, rating range, or comment keyword with pagination (public access).

8. Delete Reviews by Product (* Admin)

Endpoint: DELETE /reviews/by-product/<product_id>

Description: Deletes all reviews for a specific product (admin only).

9. Delete Reviews by User (* Admin)

Endpoint: DELETE /reviews/by-user/<user_id>

Description: Deletes all reviews by a specific user (admin only).

10. Get Product Review Stats

Endpoint: GET /reviews/stats/product/<product_id>

Description: Retrieves review statistics for a specific product (public access).

11. Get Overall Review Stats (* Admin)

Endpoint: GET /reviews/stats/overall

Description: Retrieves overall review statistics for all products (admin only).
