1. Add New Category

Endpoint: POST /categories

Description: Creates a new category with an optional image (admin only).

2. Get Category by ID

Endpoint: GET /categories/<category_id>

Description: Retrieves a specific category by its ID (public access).

3. Get Categories by Parent

Endpoint: GET /categories/parent

Description: Retrieves categories with a specified parent ID (public access).

4. Update Category

Endpoint: PUT /categories/<category_id>

Description: Updates a specific category with an optional image (admin only).

5. Delete Category

Endpoint: DELETE /categories/<category_id>

Description: Deletes a specific category (admin only).

6. Get All Categories

Endpoint: GET /categories

Description: Retrieves a paginated list of all categories (public access).

7. Search Categories

Endpoint: GET /categories/search

Description: Searches categories by name with pagination (public access).
