0. Mock Session Login

Description: Set user ID and admin status for session-based authentication (for testing purposes).

1. Add New Cart Item

Endpoint: POST /cart/items

Description: Adds a new item to the authenticated user's cart. Requires session authentication.

2. Get User's Cart Items

Endpoint: GET /cart/items

Description: Retrieves all cart items for the authenticated user. Requires session authentication.

3. Get Cart Item by ID

Endpoint: GET /cart/items/<cart_item_id>

Description: Retrieves a specific cart item by ID (owner or admin only). Requires session authentication.

4. Update Cart Item

Endpoint: PUT /cart/items/<cart_item_id>

Description: Updates the quantity of a specific cart item (owner or admin only). Requires session authentication.

5. Delete Cart Item

Endpoint: DELETE /cart/items/<cart_item_id>

Description: Deletes a specific cart item (owner or admin only). Requires session authentication.

6. Clear User's Cart

Endpoint: DELETE /cart/clear

Description: Clears all items from the authenticated user's cart. Requires session authentication.

7. Get User's Cart Stats

Endpoint: GET /cart/stats

Description: Retrieves statistics for the authenticated user's cart. Requires session authentication.

8. Get Cart Items by User (Admin Only)

Endpoint: GET /admin/cart_items/user/<user_id>

Description: Retrieves all cart items for a specific user (admin only).

9. Get All Cart Items (Admin Only)

Endpoint: GET /admin/cart_items

Description: Retrieves a paginated list of all cart items (admin only).

10. Search Cart Items (Admin Only)

Endpoint: GET /admin/cart_items/search

Description: Searches cart items by user ID or product ID with pagination (admin only).

11. Clear Cart by User (Admin Only)

Endpoint: DELETE /admin/cart_items/user/<user_id>

Description: Clears all cart items for a specific user (admin only).

12. Delete Cart Items by Product (Admin Only)

Endpoint: DELETE /admin/cart_items/product/<product_id>

Description: Deletes all cart items for a specific product (admin only).

13. Get Overall Cart Stats (Admin Only)

Endpoint: GET /admin/cart/stats

Description: Retrieves overall statistics for all cart items (admin only).

14. Get User Cart Stats (Admin Only)

Endpoint: GET /admin/cart_items/user/<user_id>/stats

Description: Retrieves statistics for a specific user's cart (admin only).
