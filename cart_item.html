Cart Items API Documentation

Interact with the Cart Items API endpoints hosted at

Back to API Testing Hub

1. Add New Cart Item

Endpoint: POST /cart/items

Description: Adds a new item to the authenticated user's cart.

2. Get User's Cart Items

Endpoint: GET /cart/items

Description: Retrieves all cart items for the authenticated user.

3. Get Cart Item by ID

Endpoint: GET /cart/items/

Description: Retrieves a specific cart item by ID (owner or admin only).

4. Update Cart Item

Endpoint: PUT /cart/items/

Description: Updates the quantity of a specific cart item (owner or admin only).

5. Delete Cart Item

Endpoint: DELETE /cart/items/

Description: Deletes a specific cart item (owner or admin only).

6. Get Cart Items by User (Admin Only)

Endpoint: GET /admin/cart_items/user/

Description: Retrieves all cart items for a specific user (admin only).

7. Get All Cart Items (Admin Only)

Endpoint: GET /admin/cart_items

Description: Retrieves a paginated list of all cart items (admin only).
