1. Add New Category Discount (Admin Only)

Endpoint: POST /category_discounts

Description: Creates a new category discount (admin only).

2. Update Category Discount (Admin Only)

Endpoint: PUT /category_discounts/

Description: Updates a category discount's details (admin only).

3. Get Category Discount by ID

Endpoint: GET /category_discounts/

Description: Retrieves a specific category discount by its ID.

4. Get Discounts by Category

Endpoint: GET /category_discounts/category/

Description: Retrieves all discounts for a specific category.

5. Get Valid Category Discounts

Endpoint: GET /category_discounts/valid/

Description: Retrieves all valid discounts for a specific category.

6. Delete Category Discount (Admin Only)

Endpoint: DELETE /category_discounts/

Description: Deletes a category discount by its ID (admin only).

7. Get All Category Discounts

Endpoint: GET /category_discounts

Description: Retrieves a paginated list of all category discounts.
