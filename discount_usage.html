1. Add New Discount Usage

Endpoint: POST /discount_usages

Description: Records a new discount usage for the authenticated user.

2. Get Discount Usage by ID (Admin Only)

Endpoint: GET /discount_usages/

Description: Retrieves a specific discount usage by its ID (admin only).

3. Get Discount Usages by Discount (Admin Only)

Endpoint: GET /discount_usages/discount/

Description: Retrieves all discount usages for a specific discount (admin only).

4. Get Discount Usages by User

Endpoint: GET /discount_usages/user/

Description: Retrieves all discount usages for a specific user (authenticated user only).

5. Delete Discount Usage (Admin Only)

Endpoint: DELETE /discount_usages/

Description: Deletes a discount usage by its ID (admin only).

6. Get All Discount Usages (Admin Only)

Endpoint: GET /discount_usages

Description: Retrieves a paginated list of all discount usages (admin only).
